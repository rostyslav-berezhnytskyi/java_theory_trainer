# Project instructions

This is a Spring Boot 3.5 Java 21 application.

Stack:
- Java 21
- Spring Boot
- Maven
- PostgreSQL
- Flyway
- Thymeleaf
- Spring Security
- Lombok

Rules:
- Keep code simple and readable.
- Add comments in English only when they help understanding.
- Do not change database schema without creating a Flyway migration.
- Do not put secrets into application.yaml.
- Use .env for secrets and local configuration.
- Do not remove existing working functionality.
- After code changes, explain changed files.
